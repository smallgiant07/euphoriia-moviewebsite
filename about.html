Reflection

Through this assignment I learnt the importance of using external stylesheets to efficiently design web pages. Also, I found that simple designs are way better in terms of functionality and display rather than fancy websites. They are also more simple to use , i.e it affects the readability of the webpage. I realized that design isn't as important as functionality and resposiveness. Importing many css sheets and/or using heavy graphics can considerably reduce the load time speed of webpages. As I completed different requirements, I understood the importance of using relative untis which contributes to the fluidity of the webpage. Further I tried to incorporate responsive design and faced various hurdles when designing specific elements because they didn't belong to any particular class. In future projects, I would make sure that I don't focus too much on designing so that I do not miss the core requirements, as well as start designing for a mobile platform first so that things become easier.
