The responsive navigation bar and gallery helps the user to open the website from any device whose maximum width is 1080px. That is all the devices whose width is below 1080 px, can load the webpage without compromising the quality as well as fuildity of the page. The code is mentioned in 2 different css files. Third party references are mentioned in the css files as well as here.
